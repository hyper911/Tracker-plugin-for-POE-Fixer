Tracker Plugin for POE-Fixer

Кратко
- Плагин `Tracker` показывает эффекты статусов и дополнительные подсказки в POE-Fixer.

Структура репозитория
- `Plugins/Tracker/` — исходники плагина (C++/VS project).
- `Resources/tracker/` — ресурсы (иконки и т.д.).

Сборка
1. Откройте `Plugins/Tracker/Tracker.vcxproj` в Visual Studio 2022 (или новее).
2. Выберите конфигурацию `Release` и целевую платформу соответствующую вашей сборке (x86/x64).
3. Постройте проект — артефакты появятся в `Plugins/Tracker/bin/Release`.

Установка
- Скопируйте полученную `Tracker.dll` (и сопутствующие файлы, если требуются) в папку плагинов POE-Fixer.

Исключения
- В репозитории исключены логи и большие бинарные артефакты. См. `.gitignore`.
